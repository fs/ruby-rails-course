# История Ruby

Ruby был написан, чтобы сделать работу программиста проще и не заботиться о сложности работы программистом. В этот кратком введении мы рассмотрим поверхностно основные функции языка, необходимые для домашнего задания.


* История Ruby
  * balance of functional programming with imperative programming
  * more powerful than Perl, and more object-oriented than Python
  * programmer happiness is the goal
  * Influenced by:
    * Perl
    * Smalltalk
    * Eiffel
    * Lisp
    * 2.0 is coming soon!
  * People
